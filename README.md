> 챌린지 달성 의지 부여 페이백 SNS 어플리케이션.

## ✨ 서비스 소개
```sh
‘윌링’은 페이백을 통해 사용자의 목표 달성 동기를 자극함으로써 사용자가 목표 달성을 할 수 있도록 돕는 서비스입니다. 현장실습 때 서비스의 비즈니스 로직을 기획하여 학과 캡스톤 때 기획한 서비스를 구현하였습니다.
안드로이드 개발 3명, ios 개발 1명이 모인 팀으로, Firebase를 이용하여 데이터 통신을 진행하였고,
사용자간 푸시 메시지 전송 서비스를 위해 Firebase Auth 와 FCM 서비스도 이용하였습니다.
현장실습과 캡스톤에서 좋은 성적을 거둘 수 있었습니다.
```
